# Javaweb音乐播放管理系统 java341

## 项目概述

本系统是基于B/S架构和MVC设计模式的音乐播放管理系统。采用前后端分离的开发模式，前端使用Vue，后端采用SpringBoot，结合Mybatis和Mysql等技术构建。

## 技术栈

- 架构：B/S、MVC
- 系统环境：Windows
- 开发环境：
  - IDEA
  - JDK 1.8
  - Maven
  - Mysql
- 技术栈：
  - Java
  - Mysql
  - SpringBoot
  - Mybatis
  - Vue

## 功能模块

### 用户模块

- 用户注册
- 用户登录

### 游客模块

- 歌曲播放器
  - 播放歌曲
  - 歌手列表
  - 歌单列表
  - 我的音乐
  - 歌曲播放
  - 歌曲收藏
  - 歌曲下载
  - 歌曲评论

### 管理员模块

- 用户管理
- 歌手管理
- 歌单管理
- 上传歌曲

## 系统角色

- 用户
- 游客
- 管理员

## 运行环境

- 操作系统：Windows
- 数据库：Mysql

## 部署步骤

1. 准备开发环境：安装IDEA、JDK1.8、Maven和Mysql。
2. 克隆或下载项目代码。
3. 导入数据库文件到Mysql。
4. 分别在前后端项目中运行构建命令。
5. 启动后端服务。
6. 启动前端服务。

## 核心亮点

- 前后端分离，便于管理和扩展。
- 采用成熟稳定的Java技术栈，保证了系统的稳定性和可靠性。
- 提供丰富的音乐管理功能，满足不同角色需求。
